# 3. Human‑Centred Evaluation Indices (Equations 1–4)
# --------------------------
Composite Human‑Centered Score (H_sim):
  H_sim = 0.45·D_sim + 0.30·C_target + 0.25·P_target

# 3.1 Cognitive Suitability Index (C_target)
C_target = 0.4·(1 − (S−1)/(S_max−1)) + 0.3·(1 − K/K_max) + 0.3·(T/T_max)
  S = average operation steps (target ≤3)
  S_max = 12 (industry benchmark)
  K = NASA‑TLX cognitive load (target = 40)
  K_max = 100
  T = automatic misoperation correction rate (target = 100%)
  T_max = 100%
  Sub‑weights: w_S = 0.4, w_K = 0.3, w_T = 0.3

# 3.2 Demand Matching Index (D_sim) – computed directly from simulation
D_sim = Σ min(P_actual,i, P_expected,i) / Σ P_expected,i · α_i
  Priority weights α_i:
    - Peak agricultural period: 1.5
    - Normal period: 1.0
    - Low‑demand period: 0.5

# 3.3 Subject Participation Index (P_target) – design target
P_target = 0.4·(E/100) + 0.3·(τ_std/τ_actual) + 0.3·(R/100)
  E = user‑defined strategy execution success rate (%)
  τ_std = 2 s (benchmark response time)
  τ_actual = actual response time (s)
  R = demand response participation rate (%)

# --------------------------
# 4. Fuzzy Logic Model (Load Prediction & Control Rules)
# --------------------------
Type: Mamdani fuzzy inference system
Input variables: 4
  - Agricultural season (categorical)
  - Time of day (categorical)
  - Farmer type (categorical)
  - Horizontal irradiance (continuous, W/m²)
Membership functions: triangular
Number of core linguistic rules: 25
Coverage rate of electricity scenarios: 95.3%
Implementation:
  - Single‑step prediction time on STM32: 186 ms
  - Peak RAM usage: 27.4 KB
  - Average R²: 0.932
  - Mean Absolute Percentage Error (MAPE): 10.8%

# Representative fuzzy rule (Table 6):
# IF Busy Season AND Morning AND High Irradiance THEN Production_High

# --------------------------
# 5. Multi‑Objective Control Strategy (Equation 5)
# --------------------------
Objective function (maximise):
  max J = 0.42·D_sim + 0.35·(E_self / E_total) + 0.23·(1 − N_cycle,eq / N_rated)

  D_sim      : demand matching index
  E_self     : self‑consumed PV energy (kWh)
  E_total    : total PV generation (kWh)
  N_cycle,eq : equivalent full charge‑discharge cycles
  N_rated    : rated cycle life of battery

Safety constraints:
  - SOC: 20% – 80%
  - Charge/discharge power: 0 – 5 kW
  - Grid voltage: 220 V ± 7%
Weights determined via gradient sensitivity analysis (12 test sets).
